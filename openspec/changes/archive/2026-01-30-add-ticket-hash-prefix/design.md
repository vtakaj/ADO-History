## Context

作業記録テーブルと対応チケット一覧は `generate-work-table` の出力であり、現在はチケット番号が数字のみで表示されている。既に出力フォーマットの整形は `lib/formatters/markdown.sh` に集中しているため、出力形式の調整は同ファイルに集約して実施できる。

## Goals / Non-Goals

Goals:
- 作業記録テーブルと対応チケット一覧で、チケット番号の先頭に `#` を付与する
- 表と一覧の表示形式を一貫させる

Non-Goals:
- チケット取得ロジックやステータス履歴の生成方法は変更しない
- 既存データの内容や ID そのものは変更しない

## Decisions

- 出力整形は `lib/formatters/markdown.sh` に限定して変更する  
  理由: 出力箇所が集約されており、他の処理に影響を与えずに表示だけを変更できるため。
- チケット番号は表示時のみ `#` を付ける  
  理由: 保存データのフォーマット変更は不要で、既存の検索や比較に影響を与えないため。

## Risks / Trade-offs

- [見た目変更のみで対応] → 既存テストの期待値が変わるため、テスト更新が必要になる
