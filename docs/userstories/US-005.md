# Export team work reports

## Story
As a project manager
I want to export the team's monthly work report in multiple formats
So that it can be used flexibly for various purposes

## Implementation status
Not supported in the current implementation (only markdown work record table generation).

## Acceptance criteria
- [ ] Export by specifying a month
- [ ] Select multiple formats (JSON, Markdown, Excel)
- [ ] Include monthly work records for all members
- [ ] Include total monthly work time for each member
- [ ] Include a ticket list (ID, title)
- [ ] Include project name and period in the filename
- [ ] Complete export within 1 minute

## Notes
- JSON format: suitable for system integrations and automation
- Markdown format: suitable for documentation and sharing
- Excel format: usable for management reports
The filename is in the form "[ProjectName](YYYYMM)work-record-detail.[ext]".
