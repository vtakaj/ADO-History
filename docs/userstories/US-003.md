# Review team monthly work summary

## Story
As a project manager
I want to review the team's monthly work time and outcomes
So that I can do monthly reporting and staffing management

## Implementation status
Not supported in the current implementation (supports only work record table generation).

## Acceptance criteria
- [ ] Show a team-wide work summary for a specified month
- [ ] Show each member's total monthly work time
- [ ] Show a list of tickets each member worked on
- [ ] Aggregate total team work time
- [ ] Include time spent in shared events
- [ ] Easily move to previous/next month summaries
- [ ] Show the summary in a print-friendly layout

## Notes
Monthly summaries are used for reporting to management, so display them in a clear, organized format. Include information that serves as indicators of each member's utilization and productivity.
